## Setup

Install poetry with:

```shell
curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python -
```

Install the project with:

```shell
poetry install
```

## Run tests

```shell
poetry run pytest
```
